# Hangman Game

## Description
This is a web-based Hangman game built using Python and Flask as part of the CodeAlpha internship. Players guess letters to reveal a hidden word before running out of attempts.

## Features
- Interactive web interface
- Random word selection
- Letter-by-letter guessing
- Win and lose messages
- Option to play the next game

## Technologies Used
- Python
- Flask
- HTML
- CSS
- JavaScript

## Project Structure
```
app.py
Hangman.py
templates/
    index.html
static/
    style.css
    script.js
```

## How to Run

1. Clone the repository.
2. Install Flask:
   ```
   pip install flask
   ```
3. Start the application:
   ```
   python app.py
   ```
4. Open your browser and visit:
   ```
   http://127.0.0.1:5000
   ```
